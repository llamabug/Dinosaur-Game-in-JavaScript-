# Dinosaur-Game-in-JavaScript-
Teaching myself JavaScript by following a tutorial (https://www.youtube.com/watch?v=4Oz34co7VLY) and then customising it and adding extra features

<h3>How to play</h3>
Press any keys (except control keys) to make the cat jump. Avoid rocks or it's game over!
<br>
[note: sometimes you may have to restart the game multiple times for it to work]

<h3>Additional Features</h3>
<ul>
  <li>Changed Theme</li>
  <li>Colour changing background</li>
  <li>Colour changing score to match the background</li>
</ul>
